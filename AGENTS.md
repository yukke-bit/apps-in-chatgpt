# ワークスペース運用メモ

このリポジトリは、ChatGPT Apps SDK 関連のアプリやサンプルを開発・検証するためのワークスペースです。

- 既存の構成や公式サンプルの設計意図を優先し、必要以上に独自構成へ寄せない。
- MCP server と widget assets は、本番運用を見据えて分離できる構成を基本にする。
- build 済み widget bundle は `assets/` に出力する。
- 追加サンプル、共通パッケージ、抽象化は、必要性が明確になるまで増やさない。
- README、仕様書、設計メモ、運用メモなどのドキュメントは、原則として日本語で書く。
- コミットメッセージは、変更内容が分かる簡潔な日本語で書く。
- GitHub などのリモートへ push する前に、必ずユーザーへ確認して許可を取る。
- 機能説明や実装状況の説明では、専門用語よりもユーザー操作ベースで「何ができるか」「押すと何が起きるか」「どこから未実装か」を先に説明する。
- 成功、未完成、見た目だけ、要修正の区別を曖昧にせず、必要な場合だけコード上の理由を短く補足する。
