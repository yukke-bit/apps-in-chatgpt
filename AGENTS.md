# ワークスペース運用メモ

このリポジトリは、公式 Apps SDK サンプルの `pizzaz_server_node` をローカルで動かすためのワークスペースです。

- まずは `pizzaz_server_node/` の Node.js MCP server を動かすことを優先する。
- ChatGPT 内で表示する Pizzaz widget は `src/pizzaz*` に置く。
- build 済み widget bundle は `assets/` に出力する。
- 追加サンプルや共通パッケージは、必要になるまで増やさない。
- README、仕様書、設計メモ、運用メモなどのドキュメントは、原則として日本語で書く。
